# アプリの説明:
学童保育施設などで、大量のおやつの在庫や賞味期限を先生たちが簡単に管理できるWebアプリケーションです。スマホでパッケージや納品書の写真を撮る（アップロードする）と、AIが画像から自動で「おやつの名前」「賞味期限」「個数」を読み取ってくれます。もちろん手入力も可能で、同じ種類のおやつは一覧画面で自動的にまとめられ、見やすく整理されます。
# 主な機能:
AI画像認識登録: Gemini AIを使って写真からおやつ情報を自動抽出（読み取れない場合は手入力を促します）
手入力登録: 写真を使わずに直接データを入力する機能
スマート在庫加算: 同じ「名前」と「賞味期限」のおやつが登録された場合、在庫データをダブらせず自動で個数を足し上げ
在庫一覧ボード: おやつの種類ごとにグルーピングし、賞味期限順に並び替えて一覧表示
ワンタッチ増減＆削除: 在庫ボードから「＋」「ー」ボタンで直接個数を調整、不要な行はインライン確認で安全に削除
先生ログイン機能: Googleアカウントを用いた安全なログインシステム
# 技術スタック (Tech Stack):
* **フロントエンド:** React (Vite), TypeScript, Tailwind CSS（スタイリング）
* **アイコン:** Lucide React
* **バックエンド / データベース:** Firebase (Authentication: Googleログイン, Firestore: リアルタイムデータベース)
* **AI・機械学習:** @google/genes (Gemini 2.5 Flash を使用し、高速かつ精度の高い画像認識とJSON構造化データ抽出を実行)

# 使い方:
1.ログインする
右上にある「先生ログイン」ボタンを押し、自身のGoogleアカウントでログインします。（在庫を保存・閲覧するために必要です）
2.おやつを登録する （2つの方法）
 * **AIにお任せ:** 「おやつを登録」タブで、枠の中にパッケージの写真をポイッと入れるか、タップして写真を選びます（スマホならカメラが起動します）。AIが一生懸命読み取って、入力欄を自動で埋めてくれます。
* **手動で打つ:** 写真を使わない場合は「自分で手入力する」ボタンを押し、名前・期限・個数を自分で打ち込みます。
内容が合っていれば「データベースに保存」を押して登録完了です！
在庫をみる・管理する
「在庫を見る」タブに切り替えると、現在のおやつが種類ごとにまとまって表示されます。
おやつを使ったり補充したときは、表の「＋」「ー」ボタンで手軽に個数を変えられます。
間違えて登録した行や、在庫がゼロになって消したい行は、右端の「ゴミ箱」アイコンを押して「はい」を選ぶと消去できます。
